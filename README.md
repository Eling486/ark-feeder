# Ark-Feeder - 鹰角小姐的发饼机

> 面对天灾，我们好活不断！

## 前期准备 - Preparation

- 一台服务器
  - 提前安装：[`Node.js`](https://nodejs.org/en/), [`Git`](https://git-scm.com/downloads)
- 注册企业微信，并新建应用

## 安装方法 - Introduction

```
git clone https://github.com/Eling486/ark-feeder.git
cd ark-feeder
npm install
```

## 使用方法 - Usage

- 提前修改`config.yaml`文件中的相关设置为你自己的企业信息

```
npm run start
```

运行正常输出：

```
2021/1/1 下午1:00:00 - 公告无更新
2021/1/1 下午1:00:00 - 官方微博无更新
```
